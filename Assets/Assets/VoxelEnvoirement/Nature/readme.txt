Troubleshooting

Blurry Textures

Issue: The textures appear blurry.
Solution:

Select the blurry texture in the Inspector.

In the Filter Mode setting, choose Point Mode (No Filter).

Shadow Projection Issues

Issue: Shadows have projection errors, such as gaps or artifacts.
Solution:

Go to your Light settings.

Reduce the Shadow Bias value to minimize projection errors.
